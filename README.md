# SEO Pulse 🚀

**SEO Pulse** is a professional, world-class Search Engine Optimization (SEO) consultant chatbot designed to help users dominate search engine results. Developed by **Muhammad Usman Umar**, this application leverages advanced AI to provide actionable, data-driven SEO strategies.

![SEO Pulse Branding](https://picsum.photos/seed/seopulse/800/400)

## ✨ Features

- **Expert SEO Advice**: Get advanced strategies for On-page, Technical, and Off-page SEO.
- **Personalized Experience**: Dynamic greetings and user-specific context.
- **Modern UI/UX**: A sleek, Emerald-themed interface designed for clarity and growth.
- **Secure Authentication**: Built-in Sign In and Sign Up flow (currently using a mock backend for demonstration).
- **Real-time Analysis**: Fast, AI-powered responses to complex SEO queries.
- **Developer Credit**: Proudly developed by Muhammad Usman Umar.

- **Frontend**: [React](https://reactjs.org/) with [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)

- **AI Engine**: [Google Gemini API](https://ai.google.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- A Gemini API Key from [Google AI Studio](https://aistudio.google.com/app/apikey)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/seo-pulse.git
   cd seo-pulse
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Build for production**:
   ```bash
   npm run build
   ```
